## 站內信匣：與 `guest_x` 的對話

介面呈現成老論壇的「悄悄話」列表：左側頭像空白，時間戳精確到分。每則訊息下方有 `已讀 / 未讀` 狀態——這是本關真正的恐怖來源。

---

### 第一段：認識（失蹤前約三週）

```
guest_x    看到你在心情板的文了。半夜睡不著的人我懂。
                                              已讀

k_r_o_w    你也睡不著喔
guest_x    嗯。你聽起來過得不太好。要聊嗎，我不太會睡。
                                              已讀

k_r_o_w    ……好啊。反正也沒人要聽。
```

> **設計註**：`guest_x` 的開場只到「我懂」「要聊嗎」為止——一個普通的陌生善意。真正讓人心疼的是 `k_r_o_w` 那句「反正也沒人要聽」，接續了舊文的孤獨。不寫任何操弄話術。

---

### 第二段：他難得放下防備（約兩週前）

```
k_r_o_w    今天又不想回家
k_r_o_w    你會不會覺得我很煩
guest_x    不會。你想待著就待著。
                                              已讀

k_r_o_w    謝謝 你是第一個這樣說的人
k_r_o_w    這裡好安靜 只有你會回我
```

> **設計註**：「只有你會回我」——玩家此刻已知道結局，這句會很痛。仍然沒有祕密指示、沒有「別跟別人說」，避免任何誘拐示範。孤立感是他「原本就」被世界略過造成的，不是被對方製造的。

---

### 第三段：那個邀約（失蹤前幾天）

```
guest_x    週末有個趴，某個大少爺辦的，場面很誇張。
guest_x    你這種聊得來的，去了一定不無聊。要來嗎？
                                              已讀

k_r_o_w    我？可以喔？
guest_x    當然。我傳地址給你。
guest_x    [附檔：party_0xx.jpg]
                                              已讀

k_r_o_w    好 我去
k_r_o_w    好久沒這麼期待一件事了
```

> **設計註**：邀約只寫到「大少爺的趴」「場面誇張」。那張 `party_0xx.jpg` 就是**通往關卡 5 的線索圖**——玩家把它下載、調亮後看到背景的側臉，檔名本身是下一關密碼。`k_r_o_w` 最後那句期待，是全劇最殘忍的一行。

### 第四段：戛然而止

```
（派對當晚 22:xx 之後，k_r_o_w 未再傳出任何訊息）

guest_x    到了嗎
                                              未讀
guest_x    人呢
                                              未讀
guest_x    ？
                                              未讀
```

> **設計註**：狀態從「已讀」變成一整排「未讀」。不需要一個字說明發生了什麼——`k_r_o_w` 再也沒有點開這些訊息。用介面說故事，勝過任何描寫。

---

### 第五段：更後面（服務真假難辨的核心）

派對三天後，`guest_x` 的語氣變了——像是有別人在用這個帳號，或同一個人開始慌了：

```
guest_x    你把我們的對話刪掉。現在。
                                              未讀
guest_x    這個帳號以後不要再上線。
                                              未讀
（此後 guest_x 未再出現）
```

然後——**兩年後**，也就是懸賞文出現的同一時期，這串對話最底下，突然多出一則：

```
k_r_o_w    我讀了。
                                              （剛剛）
```

> **設計註**：這是全關的引爆點，也是「真假難辨」的心臟。
> - **guest_x 那兩則**：「刪掉對話」「別再上線」——強烈暗示有人在善後、在滅口，支持「這是一場被掩蓋的犯罪＋有人在做局」的讀法。
> - **兩年後那則「我讀了」**：那些一直「未讀」的訊息，被讀了。是誰讀的？支持「他還在線上」的讀法。
> - 遊戲**不解答**。玩家看到「（剛剛）」這個時間戳，會頭皮發麻——然後自己嚇自己。

## 本關的線索交付

1. **情感**：確立 `guest_x` 這條線與 `k_r_o_w` 的孤獨，讓後面的反轉有重量。
2. **物證**：`party_0xx.jpg` → 交給關卡 5（調亮看側臉、檔名解密）。
3. **懸念**：「刪掉對話」暗示善後者存在（指向後面的政治人物父親）；「我讀了」把鬼的可能性重新丟回桌上。

## 實作小技巧

- `已讀 / 未讀` 狀態建議做成視覺上明顯的差異（灰字／紅字），讓「一整排未讀」有壓迫感。
- 最後那則「我讀了」可以做成**玩家停留在此頁一段時間後才淡入出現**（JS 計時），製造「它是趁我在看的時候出現的」錯覺。
- `guest_x` 全程不給真實身分，IP／註冊資料留白，維持「可能是兇手陣營、也可能只是個中間人」的模糊。
