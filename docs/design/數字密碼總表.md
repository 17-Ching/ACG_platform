固定用這組值,實作時照抄:

| 元素 | 值 | 來源 |

| 照片內電子鐘 | `23:47` | 調亮後牆上時鐘顯示 |

| 運算規則 | 逐位不進位相加 | 老 BBS 整人手法 |
| 計算 | `347 + 831`：(3+8=11→**1**)(4+3=**7**)(7+1=**8**) | |
| **結果路由** | **`/thread/178`** | 玩家填入隱藏頁網址 |

| 段 | 值 | 來源 |
|---|---|---|
| 第 1 段 | `831` | 別墅座標尾三碼(關卡 5) |
| 第 2 段 | `0314` | 站內信通行碼 / 生日(關卡 3) |
