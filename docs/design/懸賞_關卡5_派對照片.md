# 《懸賞：ID 還在線上》— 關卡 5：派對照片（完整謎題與解答）

> 承接關卡 4：玩家在站內信拿到 `guest_x` 傳的附檔 `party_0xxx.jpg`。這一關要玩家把「一張看似普通的模糊照」拆成三層線索，最後把矛頭第一次指向「大少爺」。
> 難度：中等偏上（本作解謎的高峰之一）。三層：**調亮看內容 → EXIF 對出矛盾 → 檔名解密進下一層**。

---

## 一、這張照片長什麼樣

表面上：一張手機隨拍的派對照，很暗、很糊。看得到杯子、彩色燈光、幾個模糊人影。乍看沒什麼。

實際上藏了三層資訊，對應三個解法步驟。

---

## 二、第一層：調亮 → 看見側臉

照片刻意壓得很暗。玩家要想到「把它調亮／下載後拉曝光」。

- **實作**：提供可下載的原圖，或在頁面放一個「亮度」滑桿（前端 CSS `filter: brightness()`）。
- **調亮後**：畫面右後方浮現一個原本看不清的**側臉**——衣著講究、手上戴著一只有特徵的錶／戒指。這個人明顯不屬於「一個逃家少年的世界」。
- **交叉點**：這張側臉，與八卦板正在被熱議的「某政治世家大少爺」的公開照片，特徵對得上（同一只錶／同一個側臉痣／同一件限量單品）。玩家自己把兩張圖並在一起時，寒意就上來了。

> **設計提示**：不要讓側臉「明講是誰」。讓玩家自己在八卦板找到對照、自己下結論。玩家親手拼出來的比你告訴他的可怕十倍。

---

## 三、第二層：EXIF → 時間與地點的矛盾

引導玩家檢視圖片資訊（可做成頁面上的「內容資訊」按鈕，或讓玩家真的下載看 EXIF）。裡面藏兩個矛盾：

1. **拍攝時間**：`party_0xxx.jpg` 的拍攝時間，落在 `k_r_o_w` **官方失蹤日的當晚**——也就是說，他失蹤那天人根本在這場派對。
2. **GPS 座標**：座標指向一處私人別墅／招待所，而非任何官方說法提到的地點。這個地址在關卡 7 會再出現，與「被抹掉的紀錄」對上。

- **實作**：EXIF 可以是真的寫進圖檔（進階玩家用工具看），也可以做成頁面上一個模擬的「相片詳細資訊」面板（大眾玩家看得到）。建議兩者都給，照顧不同玩家。

> **雙關設計**：這些 EXIF 是「證據」，但也可以被讀成「有人故意留的餌」——一張證據完整到剛剛好的照片，究竟是死者拚了命留下的，還是有人精心佈的局？繼續不解答。

---

## 四、第三層：檔名解密 → 下一關入口

檔名 `party_0xxx.jpg` 的 `0xxx` 不是亂碼，是密碼。

### 謎題機制：時間 + 座標整數相加

- 照片裡電子鐘的「時、分」，加上 EXIF GPS 座標的「整數部分」（北緯、東經各取整數），四個數單純相加。
- 檔名 `0xxx` 暗示「四碼、首位是 0」→ 把算出的三位數前面補 0 湊四碼。
- 解出的四碼是**八卦板那篇上鎖文章的解鎖碼**，解開即進入**關卡 6：人肉搜索反轉**。

### 一個具體可實作的版本（含解答）

為了讓你直接能做，給一版明確的（數值一律以《數字密碼總表》為準）：

1. 檔名 `party_0xxx.jpg` 提示玩家「缺四碼，首位是 0」。
2. 調亮後的照片裡，牆上電子鐘顯示 `23:47` → 時 `23`、分 `47`。
3. EXIF 的 GPS 座標整數部分是北緯 `24`、東經 `121`。
4. 規則（提示只暗示「時、分、座標的整數，加起來」，不明寫算式）：
   - `23 + 47 + 24 + 121 = 215` → 依檔名補 0 → **`0215`**
5. 用 `0215` 解開八卦板的上鎖文章（`/thread/256`）→ 進入關卡 6。

> 單純相加＋補 0 湊碼，中等難度、不需外部工具、驗算得出來。只要**線索都在照片與 EXIF 裡自足**即可。

### 實作與防呆

- 網址錯誤時，回一句 in-world 的話（例如頁面顯示「文章不存在。……還是你根本不想找到？」），維持毛感也提示玩家再算一次。
- 若怕玩家卡關，可讓 KKcat 在懸賞串裡丟一句軟提示：「檔名從來不是隨便取的。時間，還有他傳給你的座標。」——順便強化 KKcat 「懂他」的人設。

---

## 五、本關的線索交付

1. **指認**：第一次把矛頭指向「大少爺」（但由玩家自己拼出，非明講）。
2. **時空定位**：EXIF 把 `k_r_o_w` 的失蹤日與這場派對、這個私人地點釘在一起——關卡 7 的「被抹掉的紀錄」就從這個地址與日期展開。
3. **通道**：檔名密碼 `0215` → 解開八卦板上鎖文章（`/thread/256`）→ 關卡 6 人肉搜索反轉。
4. **曖昧延續**：這張「剛好完整」的照片，是遺留還是誘餌？鬼與局的拉扯不斷線。

---

## 六、給創作者的提醒

- 照片、座標、人物全為虛構；「大少爺」與其家世請完全架空，**不影射任何真實政治人物或家族**。
- 派對畫面只到「燈光、杯子、人影、一個不該在場的側臉」為止，**不描寫藥物、不描寫過程**——維持全作的留白分寸。
- 側臉的「指認」交給玩家自己完成，設計上不寫死是誰，既更毛也更安全。
