# 《懸賞：ID 還在線上》— KKcat 情感線 & 終局收束

> KKcat 是全劇唯一「一直記得 `k_r_o_w`」的角色。他的作用是給這個冰冷的故事一個溫度，也是兩個結局的情感重量來源。
> 設計原則：KKcat 這條線要**同時能餵給「鬼」與「局」兩種讀法**，並且在玩家做出最終選擇時，讓代價變得具體、有臉、有名字。

---

## 一、KKcat 是誰（定位）

一個普通的老論壇網友。當年在 `k_r_o_w` 的貼文裡，他是**唯一一個問出「發生什麼事了嗎」的人**（舊文②），也是 `k_r_o_w` 失蹤前那句「幫我跟 KKcat 說聲謝謝」的對象（舊文⑤）。

他不是偵探、不是記者、沒有超能力。他只是一個**沒能接住朋友、兩年來一直過不去的普通人**。他的動機從頭到尾只有一個：

> 「我那時候就差一點點，就問出來了。」

這份「差一點」的愧疚，是整條線的核心情緒，也呼應舊文②裡那個被「沒事啦 想太多」擋掉的、差點成立的救援。

## 二、KKcat 在遊戲中的存在方式

他不是一次性 NPC，而是**貫穿全程的旁觀者兼良心**：

- **懸賞串裡**：當其他網友只想搶懸賞金、講風涼話時，KKcat 是唯一認真的人。他會在關鍵樓層留下提醒（「這個帳號我認識，他不是這種人」「你們找的不是寶，是一個人」）。
- **他會擋在玩家前面**：越接近真相，KKcat 越緊張。他既想要真相曝光，又怕玩家（跟他一樣）出事。這種矛盾讓他有血肉。
- **他掌握玩家沒有的記憶**：某些線索只有 KKcat 能補完（例如 `k_r_o_w` 提過的某個細節），讓玩家必須「跟他合作」而不只是利用線索。

## 三、中段的曖昧反轉（服務真假難辨）

玩家玩到一半，會自然生出一個念頭：**懸賞文會不會是 KKcat 發的？**

線索兩面倒：

| 指向「是 KKcat 做的局」 | 指向「不是他」 |
|---|---|
| 他對 `k_r_o_w` 的事知道得太多、太細 | 他被問到時真的一臉錯愕 |
| 藏頭密碼那種「只有懂他的人才排得出來」的手法 | 他的帳號活躍時間與懸賞文操作時間對不上 |
| 他比誰都急著要真相曝光 | 他也在害怕，不像掌控全局的人 |

**遊戲不給答案。** 兩種讀法各自成立：
- 若是 KKcat 做的 → 這是一個活人用盡辦法替死去的朋友討公道，「鬼發文」只是他借的殼。溫柔到令人心碎。
- 若不是 KKcat → 那到底是誰把帳號弄上線的？「他還在線上」的可能性又回來了。

> **關鍵**：讓玩家在「這是一個好人的執念」與「這真的是他的鬼」之間反覆——這正是本作要的餘韻。

## 四、終局：KKcat 讓兩個結局有了代價

到終局，那筆「封口費」匯進來、附上「刪掉你手上的東西」的訊息時，KKcat 也在線上。他知道玩家手裡有能扳倒大人物的證據。他傳來最後一則私訊——**然後把選擇權交給玩家**：

```
KKcat    我沒辦法替你按下去。
KKcat    但拜託你……不要再讓他一個人。
```

玩家的選擇不再只是「揭發 vs. 拿錢」，而是「要不要辜負這個等了兩年的人」。

---

### 結局 A｜揭發

玩家公開證據。新聞跑馬燈開始滾動（結果仍留白，正義是否真的到來不明說）。然後：

```
KKcat    …謝謝你。
KKcat    我終於可以跟他說，有人聽見了。
```

畫面最後，`k_r_o_w` 的帳號狀態從「在線上」變成「離線」——
- 讀成「鬼」：他等到有人替他說話了，終於能走。
- 讀成「局」：只是有人登出了。

KKcat 的帳號也隨後安靜下來，簽名檔悄悄改成一句：「你不是沒人記得。」
這是給玩家、也是給那個孩子的告別。**這是溫暖但不廉價的結局——沒人保證大人物真的伏法，但至少他不再是無人聞問。**

---

### 結局 B｜收封口費

玩家刪除證據、收下錢。畫面安靜下來，錢到帳，一切歸於平常。KKcat 的最後一則訊息停在「未讀」：

```
KKcat    你還在嗎
                                    未讀
KKcat    是不是……連你也
                                    未讀
```

（呼應關卡 4 `guest_x` 那排未讀——玩家成了又一個讓他「已讀不回」的人。）

過一陣子，論壇上出現**另一個沉寂帳號的新懸賞文**，開頭一字不差。循環暗示：局還在轉，而你這次站在了掩蓋的那一邊。

KKcat 從此再沒上線。他的最後一篇公開發文只有一句：

```
KKcat    原來每次都是這樣。
```

**這是最冷的結局——不是被大人物打敗，而是連唯一還相信的人，都被你證明了「果然沒人會為他堅持到底」。**

---

## 五、進階彩蛋（可選，給想更毛的創作者）

如果想在最深處再埋一層真假難辨：讓細心的玩家在人肉 KKcat 帳號時發現——**KKcat 的活躍時間與 `k_r_o_w` 從不重疊，寫作習慣（標點、慣用詞）又詭異地相似。**

留一個永遠不解答的疑問：他們真的是兩個人嗎？還是那個太孤獨的孩子，一直用兩個帳號在跟自己說話、假裝有人在乎？

- 這個彩蛋只給願意挖到底的玩家，主線不強迫看到。
- 一旦種下，玩家回頭重讀所有舊文都會不一樣——那份孤獨會冷到骨子裡。
- 依然不給答案。這是本作「真假難辨」的最終形態。

> 使用建議：這層很重，會把故事推向極度悲傷。若你想讓 KKcat 單純當溫暖的支點，就**不要**放這個彩蛋。兩種取向都成立，看你要故事最後留給玩家的是「一點光」還是「一片寒」。

---

## 給創作者的提醒

- KKcat、帳號與對話全為虛構，請勿影射真實個案。
- KKcat 這條線的力量在於「差一點就能救到人」的愧疚，與體制／人情冷漠的對照——維持這個基調，不要把沉重段落具體化。
- 兩個結局的代價都透過 KKcat 具象化，建議在片尾（尤其結局 B 之後）附上少年與心理求助管道連結，給玩家一個溫柔的落點。
