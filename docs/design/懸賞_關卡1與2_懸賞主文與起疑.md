# 《懸賞：ID 還在線上》— 關卡 1 & 2：懸賞主文 & 起疑

> 這是玩家進站後看到的「第一畫面」與「第一次起雞皮疙瘩」。主設計文件裡只寫了機制，這份把它變成實際的論壇文本：置頂懸賞主文、網友跳坑的回文串、個資頁的時間悖論，以及關卡 2「回到我開始的地方」怎麼把玩家導向最早那篇舊文。

---

## 關卡 1｜懸賞主文（玩家的第一畫面）

置頂在「都市傳說板」或「協尋板」。標題與內文要營造「半信半疑但忍不住點進來」的鉤子。

```
[置頂][懸賞] 我藏了東西。找到的人，錢是你的。
作者  k_r_o_w
時間  （今天）00:00
────────────────────────────────

規則很簡單：跟著線索走就好。

第一關免費送你：回到我開始的地方。
那裡有人替我留了話，只是沒人看得懂。

找對了，你就知道下一步。找到最後，
你會知道一個很有頭有臉的人，不想讓你知道的事。

懸賞金：夠你不用再上班。
真的。到帳那天你就信了。

—— 別問我是誰。你遲早會查到。

推 rich_hunter：又一個釣魚文 zzz
推 rich_hunter：不過懸賞金多少先講
推 momo_2：k_r_o_w？這 ID 我好像有印象欸
推 KKcat：……這個帳號我認識。他不是會開這種玩笑的人。
推 KKcat：而且他兩年沒上線了。
推 路人：樓上認真的？起雞皮疙瘩
推 abc999：反正閒著，我玩玩看。回到我開始的地方是三小
推 momo_2：他自己最早的文？點他 ID 看發文紀錄啊
```

> **設計註**：
> - 懸賞金**故意不寫金額**，用「夠你不用再上班」「到帳那天你就信了」吊胃口——也為終局「真的匯進一筆錢」埋伏筆。
> - KKcat 一登場就定調：他認識這個人、而且「兩年沒上線」——把時間悖論的種子埋下，接關卡 1 的個資頁。
> - 最後由網友 `momo_2` 把玩家推向正確操作（點 ID 看發文紀錄），自然引導、不出戲。

## 關卡 1｜個資頁的違和（第一次起疑）

玩家點 `k_r_o_w` 的 ID → 個人資料頁。這裡藏第一個「說不出哪裡怪」的點。

```
會員資料 —— k_r_o_w
────────────────────────────────
註冊日期    （三年前）
發文總數    17
生日        3 / 14
最後上線    （今天）00:03    ← 玩家會先略過，之後才驚覺

簽名檔：
「如果我之後都沒上線，幫我跟他說聲謝謝。」
```

> **設計註**：
> - 「最後上線（今天）」現在看沒事,但玩家玩到關卡 6（人肉出他兩年前就失蹤）再回來看這一行,會頭皮發麻——**同一個畫面,前後兩種溫度**。這是本作最愛用的手法。
> - 生日 `3 / 14` 就是關卡 3 站內信通行碼 `0314` 的來源,先大方擺在這裡,玩家當下不會注意,解到藏頭「生日就是密碼」時才回頭來找。
> - 簽名檔正是舊文⑤的最後一句(給 KKcat 的話),把懸賞主文、個資頁、舊文三處用同一句話縫在一起。

## 關卡 2｜「回到我開始的地方」→ 最早的貼文

懸賞主文說「回到我開始的地方」。玩家在個資頁看發文紀錄,一路往下翻到**最早那篇**(舊文①)。

### 呈現方式

發文紀錄頁把 17 篇文由新到舊列出。最舊那篇要讓玩家「翻到底」才看到——製造「越往下越冷」的體感:

```
發文紀錄 —— k_r_o_w（由新到舊）
────────────────────────────────
· [懸賞] 我藏了東西…            （今天）
────────────────────────────────
        （中間空白兩年，沒有任何發文）
────────────────────────────────
· （無標題）                     兩年前 · 失蹤當週
· [心情] 認識了一個網友…         兩年前
· [閒聊] 今天生日                兩年前
· [問卦] 你們家晚上會很安靜嗎    兩年前
· [心情] 有人半夜也睡不著的嗎    兩年前 ← 最早的一篇
```

> **設計註**:那段「空白兩年,沒有任何發文」用視覺化的斷層呈現——玩家會直覺意識到「這個人消失了兩年,然後突然回來發懸賞」。不需一句解釋,斷層自己在說話。

### 到站後的第一個真正線索

玩家點進最早那篇(舊文①「有人半夜也睡不著的嗎」),看到內文與那六則「路過灌水」推文。懸賞主文說的「**有人替我留了話,只是沒人看得懂**」,指的就是這六則推文的藏頭——直接把玩家交棒給**關卡 3**(解藏頭「生日就是密碼」→ 回個資頁拿生日 `0314` → 開站內信)。

## 雙關設計（服務真假難辨）

從第一畫面就要讓兩種讀法並存:

| 讀成「鬼」 | 讀成「局」 |
|---|---|
| 沉寂兩年的死者帳號自己上線發文 | 有人拿到帳號密碼、重啟這個 ID |
| 簽名檔像遺言,像本人 | 「別問我是誰」像刻意經營的懸念 |
| 最後上線「今天」——他還在 | 最後上線「今天」——有人剛登入過 |

懸賞主文那句「別問我是誰。你遲早會查到」是雙關的關鍵:既像死者賣關子,也像佈局者留線。**遊戲不解答。**

## 線索交付總表(關卡 1 → 2 → 3)

1. **關卡 1 懸賞文**:給任務、給懸賞金伏筆、KKcat 定調、`momo_2` 導向操作。
2. **關卡 1 個資頁**:埋「最後上線」時間悖論(關卡 6 引爆)、生日 `0314`(關卡 3 用)、簽名檔縫合三處。
3. **關卡 2 發文紀錄**:「空白兩年」的視覺斷層 → 翻到最早貼文。
4. **關卡 2 → 3**:最早貼文的六則推文藏頭 = 關卡 3 起點。

## 給創作者的提醒

- 帳號、內文、金額全為虛構,請勿影射真實個案。
- 「有頭有臉的人」在關卡 1 只用抽象詞帶過,不點名、不影射真實人物,把指認留到後面由玩家自己拼。
- 沉重的背景(逃家、失蹤)靠「空白兩年」「簽名檔」等留白呈現,維持全作分寸。
