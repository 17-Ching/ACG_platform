# 《懸賞：ID 還在線上》— 關卡 7：被抹掉的紀錄

> 承接關卡 6。玩家已經拼出「發懸賞的帳號=兩年前失蹤的孩子」。這一關要回答下一個問題:**為什麼一個孩子失蹤,會「不了了之」?** 答案不是沒人查,而是**有人有能力讓他從系統裡消失**——矛頭第一次正式轉向政治人物父親。
> 難度:中等。玩家不解碼,而是**交叉比對三份日期兜不攏的紀錄**,自己推出「有人動過手腳」。
> 恐怖來源:官僚式的冷感。一個孩子被幾行「已結案」「查無資料」抹掉,比任何血腥都冷。

---

## 一、這一關怎麼進行

關卡 6 留下兩個追查點:那則「不了了之」的地方新聞,和關卡 5 EXIF 裡的**別墅座標＋失蹤日**。這一關讓玩家把散落各處的官方紀錄調出來並排,發現它們**彼此矛盾**。

線索散在論壇不同角落(有人轉貼、有人截圖),玩家要自己收集齊、自己拼。

---

## 二、三份兜不攏的紀錄(核心線索)

### 紀錄 A：地方新聞(關卡 6 出現過)

```
[轉貼] 兩年前的舊聞,有人記得嗎
────────────────────────────────
〈少年離家後失蹤 家屬報案〉
……一名十四歲少年離家後失去聯繫,家屬於數日後報案。
警方表示已受理協尋。……

（下方留言僅一則）
└ 這則後來沒有後續了。就這樣沒了。
```

### 紀錄 B：協尋案件的「結案」截圖

```
[截圖] 我調到這個案子的狀態,你們看日期
────────────────────────────────
案件狀態    已結案
結案事由    當事人自行返家,協尋解除
結案日期    ← 注意這個日期
```

### 紀錄 C：關卡 5 的派對照 EXIF(玩家手上已有)

```
party_0xx.jpg
拍攝時間    ← 就是失蹤日當晚
GPS 座標    ← 指向某私人別墅／招待所
```

---

## 三、玩家要拼出的矛盾(這一關的「解」)

把三份並排,時間線就崩了:

1. **紀錄 B 說**「當事人自行返家,協尋解除」——結案日期在失蹤後沒幾天。
2. **紀錄 C 說**：他失蹤日當晚人在那場別墅派對,之後再沒任何上線、發文、訊息已讀(關卡 4 的一整排未讀)。
3. **推論**:一個「已讀不回、帳號沉寂兩年、很可能不在了」的人,官方紀錄卻寫他「自行返家」平安無事。

> **玩家自己得出的結論**:這個案子不是查不到,是**被人快速壓下、草草結掉**。而「自行返家」這種結案事由,正好是對付「一個沒人會追問的孩子」最省事的說法。

- **雙關維持**:這也可以被讀成「有人佈局時故意讓玩家發現這個矛盾」。證據太乾淨,究竟是真相浮現,還是被引導?繼續不解答。

---

## 四、矛頭轉向:從別墅到父親

玩家把「別墅座標」丟回論壇查,八卦板早有人聊過這處地點——它屬於某個**政治世家**。再把關卡 5 那張側臉(大少爺)接上:

```
八卦板舊文串（玩家自己翻到）
────────────────────────────────
· 那棟別墅是 X 家的招待所啦,誰不知道
· X 家那個小開最愛在那邊開趴,誇張到鄰居檢舉過
· 他老爸不是正要選（某個位子）嗎,最近超愛做形象
· 噓 別亂講 那家人你惹不起
```

玩家把三件事並在一起——**失蹤日的別墅派對(兒子)＋光速結案的協尋(有力人士才辦得到)＋正在衝形象的父親**——自己推出結論:

> 出事的是兒子,擦屁股、讓孩子從系統裡消失的,是那個「正要更上一層樓、輸不起」的父親。

- **設計提示**:全程**不點名真實人物**,`X 家`保持代號。指認由玩家親手完成,既更毛也更安全。

---

## 五、這一關的情緒:冷感即恐怖

不要出現屍體、不要出現犯罪過程。這一關最冷的畫面是那行公文:

```
結案事由    當事人自行返家,協尋解除
```

一個孩子的消失,被一句制式的「自行返家」輕輕蓋過。體制運轉如常,只有論壇上幾個陌生人還記得他。**這種「世界毫不在意」的寒意,是整個遊戲真正的恐怖,也是它的社會重量所在。**

KKcat 在這一關的一句話可以定錘:

```
推 KKcat：「自行返家」。
推 KKcat：他要是返家了,還會叫我不要找他嗎。
推 KKcat：他根本沒有家可以回。
```

---

## 六、線索交付(關卡 7 → 終局)

1. **拼出加害結構**:兒子(派對／用藥)＋父親(權勢／湮滅)——由玩家自三份矛盾紀錄與八卦板自行組成。
2. **證據成形**:派對照(EXIF 定時定點)＋結案截圖(日期矛盾)＋站內信(關卡 4)＋人肉包(關卡 6)——這些加起來,就是終局那份「能扳倒大人物的證據包」的內容物。
3. **動機閉環**:父親「正要選、輸不起、在做形象」→ 解釋了他為何要讓一個孩子徹底消失,也鋪好終局「封口費」的來由。
4. **交棒終局**:玩家握有證據 → 進入最終選擇(揭發 / 收封口費),KKcat 情感線在此收束(見《KKcat 情感線與終局》)。

---

## 七、給創作者的提醒

- 政治世家、別墅、案件、新聞全為虛構,`X 家`請完全架空,**不影射任何真實人物、家族或案件**。
- 用藥致死與湮滅過程一律不描寫;這一關只呈現「紀錄上的矛盾」與「官僚冷感」,不呈現任何犯罪操作細節。
- 加害結構的「指認」交給玩家從公開線索自行拼出,設計上不寫死,維持全作留白與安全分寸。
